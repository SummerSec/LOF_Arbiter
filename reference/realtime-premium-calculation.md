# 实时估算净值与溢价率手算指南

> 集思录已不再显示实时溢价，券商 APP 也普遍取消实时估值。溢价率 **可以自己算**。
>
> 本文：从「实时估算净值」到「溢价率」的 **全网最清楚版** 手算流程，含国内 T+1 LOF 与 QDII 两类公式。
>
> 跨境 QDII 的 T-2 锚定、港美时差等深层逻辑见 [qdii-premium-logic.md](qdii-premium-logic.md)。

---

## 一、为什么要算「实时估算净值」？

| 现象 | 原因 |
|------|------|
| 当天净值晚上才公布 | 普通 LOF 收盘后 T+1 更新 |
| QDII / 商品 LOF 盘中看不到「真实净值」 | 净值滞后海外或期货标的 1～2 天 |
| 集思录 / 券商不再给实时溢价 | 只能 **用跟踪标的涨跌幅自己估** |

**核心思路**：用「昨日（或 T-1）官方净值 + 跟踪标的今日涨跌」还原 **此刻基金大概值多少钱**，再和场内现价比，得到溢价率。

---

## 二、国内 T+1 LOF / 商品 LOF（基础公式）

适用于：**净值 T+1 更新、跟踪国内期货/指数** 的 LOF（如部分商品 LOF）。

### 公式（背下来）

```
实时估算净值 = 昨日净值 × (1 + 跟踪标的涨跌幅 / 100)

溢价率(%) = (场内实时价 - 实时估算净值) / 实时估算净值 × 100
```

- **昨日净值**：前一交易日基金公司公布的单位净值（东方财富 / 天天基金 / 基金官网，通常前一晚 20:00 后更新）
- **跟踪标的涨跌幅**：与基金合同一致的指数或 **主力合约** 当日涨跌幅（%）
- **场内实时价**：券商 APP 现价

> 学会这一步，已超过大部分只会看（现已消失的）APP 实时溢价的人。

## 三、实操：国投白银 LOF（161226）

以下按 **沪银主力合约** 演示手算流程（教学示例；具体跟踪标的以基金合同为准）。

### 3.1 取 3 个数据（东方财富均可查）

| 数据 | 示例值 | 说明 |
|------|--------|------|
| T-1 日净值 | **2.3194** | 3 月 17 日净值 |
| 跟踪标的涨跌幅 | **-2.37%** | 沪银主力合约当日涨跌 |
| 场内实时价 | **2.914** | 券商 APP 现价 |

### 3.2 算实时估算净值

```
实时估算净值 = 2.3194 × (1 - 2.37%)
             = 2.3194 × 0.9763
             ≈ 2.2644
```

3 月 18 日晚公布的实际净值 **2.2742**，误差约 **0.5%**——手算估算可接受。

### 3.3 算溢价率

```
溢价率 = (2.914 - 2.2644) / 2.2644 × 100%
       ≈ +28.69%
```

**解读**：场内价远高于估算净值 → **大幅溢价**；若此时追高买入，需承担溢价回落风险。

## 四、QDII LOF：公式多一项汇率

以 **华宝油气 LOF（162411）** 为例：跟踪美股 **XOP**（标普石油天然气上游指数），股票仓位约 **95%**，净值滞后美股约 1 天。

### 4.1 取 4 个数据

| 数据 | 说明 |
|------|------|
| **T-1 日净值** | 基金公司前一晚公布（如 3/18 盘中用 3/17 晚净值） |
| **XOP 实时涨跌幅** | 美股盘前/盘中 XOP 最新涨幅（%） |
| **USD/CNY 实时涨跌幅** | 美元兑人民币汇率涨跌幅（%） |
| **场内实时价** | 券商 APP 搜 162411 |

### 4.2 估算公式（A 股盘中）

```
实时估算净值 = T-1日净值 × (1 + XOP涨跌幅 × 0.95 / 100) × (1 + USD/CNY涨跌幅 / 100)
```

| 参数 | 含义 |
|------|------|
| **0.95** | 股票仓位约 95%，约 5% 现金可忽略 |
| 汇率项 | 波动通常 ±0.1% 量级；可简化省略，误差略增 |

**简化版**（汇率波动很小时）：

```
实时估算净值 ≈ T-1日净值 × (1 + XOP涨跌幅 × 0.95 / 100)
```

### 4.3 数据去哪找

| 数据 | 来源 |
|------|------|
| T-1 日净值 | 华宝基金官网、天天基金、东方财富 |
| XOP 涨跌幅 | 新浪财经、雪球、富途、老虎（搜 XOP） |
| USD/CNY 涨跌幅 | 新浪外汇等行情页 |
| 场内实时价 | 券商 APP |

### 4.4 演算示例（3 月 18 日）

**输入：**

- T-1 净值 = **0.898**
- XOP 涨跌幅 = **+1.62%**
- USD/CNY 涨跌幅 = **-0.18%**
- 场内价 = **0.894**

**估算净值：**

```
= 0.898 × (1 + 0.0162 × 0.95) × (1 - 0.0018)
≈ 0.898 × 1.01539 × 0.9982
≈ 0.9102
```

当晚公布的净值（3 月 17 日）**0.9108**，误差约 **0.1%**。

**溢价率：**

```
= (0.894 - 0.9102) / 0.9102 × 100%
≈ -1.78%
```

**解读**：折价约 1.78%，场内价低于估算净值。

> QDII 若需更严谨，应使用 **T-2 锚定 + R(T-1)** 模型，见 [qdii-premium-logic.md](qdii-premium-logic.md)。本节「T-1 净值 × 标的涨跌」为 **盘中快速手算** 常用近似。

## 五、公式对照速查

| 基金类型 | 实时估算净值 | 溢价率 |
|----------|--------------|--------|
| **国内 / T+1 LOF** | 昨日净值 × (1 + 标的涨跌幅%) | (现价 - 估算净值) / 估算净值 × 100% |
| **QDII（完整）** | T-1净值 × (1 + 标的% × 仓位) × (1 + 汇率%) | 同上 |
| **QDII（简化）** | T-1净值 × (1 + 标的% × 仓位) | 同上 |
| **QDII（严谨）** | N(T-2) × (1 + R(T-1) × 仓位) × (1 + 汇率%) | 见 QDII 专文 |

## 六、常见误区

| 误区 | 正解 |
|------|------|
| 用「最新官方净值」直接算溢价 | QDII/商品 LOF 官方净值 **滞后**，需用估算净值 |
| 相信小众网站的「实时溢价」 | 数据源不一，有时 **错得离谱**；自己按公式算最可靠 |
| 美股 QDII 看 APP「次日预估溢价」下单 | 次日无法精准，见 [qdii-premium-logic.md](qdii-premium-logic.md) |
| 161226 等商品 LOF 跟踪标的搞错 | **以基金合同为准**；手算前确认是沪银、COMEX、SLV 还是其他 |

## 七、与 LOF Arbiter 的对应

LOF Arbiter 在 ETL 中 **自动完成** 上述计算（配置见 `data/fund_tracking_config.json`）：

| 手算步骤 | 系统模块 |
|----------|----------|
| 取 T-1 / T-2 净值 | `scripts/etl.py` → 东方财富净值 |
| 取跟踪标的涨跌幅 | `scripts/tracker.py` → akshare |
| 取汇率涨跌幅 | `scripts/tracker.py` → QDII 专用 |
| 估算净值 + 溢价率 | `scripts/estimator.py` |
| 查询 / 日报 | `scripts/query.py`、`scripts/notify.py` |

**估算方式标记：**

| 标记 | 含义 |
|------|------|
| `TRACKING_HK` / `TRACKING_US` / `TRACKING_DOM` | 有跟踪配置，完整公式 |
| `LEGACY` | 无基准数据，降级为 (现价 - 官方净值) / 官方净值 |
| `NONE` | 无净值，无法计算 |

用户问「怎么手算溢价」「集思录不显示了怎么办」时，可 **先讲本文公式**，再说明 Arbiter 已自动化并标注置信度。

## 八、一句话总结

1. **实时估算净值** = 昨日（或 T-1）净值 × 跟踪标的涨跌（QDII 再 × 汇率）
2. **溢价率** = (场内价 - 估算净值) / 估算净值
3. 集思录/券商不显示时，**自己手搓比瞎信第三方更靠谱**
4. QDII 深层逻辑与港美差异 → [qdii-premium-logic.md](qdii-premium-logic.md)
