> 场内 LOF/ETF 溢价套利的底层原理。回答用户 QDII 溢价、时差、港美差异、避坑问题时优先参阅本文。

## 一、核心概念：溢价与折价

场内交易的 QDII 基金（LOF/ETF）有两个价格：

| 价格 | 含义 |
|------|------|
| **场内市价** | 股票账户实时买卖价，由市场供需决定 |
| **基金净值** | 基金真实资产价值，由海外持仓涨跌决定 |

由此产生两种状态：

- **溢价**：市价 > 真实净值 → 买贵了，性价比低，谨慎入场
- **折价**：市价 < 真实净值 → 买便宜了，存在套利空间，适合低吸

### 通用核心公式

```
溢价率 = (场内现价 - 预估基准净值) / 预估基准净值 × 100%
```

- 正数 = 溢价
- 负数 = 折价

看似简单，但 **港股 QDII 与美股 QDII 的计算精度天差地别**——这是绝大多数散户的知识盲区，也是盲目跟风 APP「实时溢价」追高的根源。

## 二、QDII 致命规则：净值天然滞后

国内 A 股交易日为 **T 日**，但 QDII 受结算、时差、休市规则限制：

**T 日能看到的官方净值，反映的是 T-2 日（或更早）的陈旧持仓数据。**

没有任何软件可以直接给出 T 日真实净值；**所有盘中溢价，全部是人工或系统估算值。**

### 完整估值时间线（必记）

| 时点 | 含义 |
|------|------|
| **T-2 日** | 基金公司公布精准官方净值 N(T-2)，**唯一真实锚点** |
| **T-1 日** | 海外市场涨跌落地，需用指数/ETF 手动估算，无官方披露 |
| **T 日** | A 股正常交易；盘中场内价波动；海外当日涨跌影响 **下一交易日** 净值 |

**结论**：基于 T-2 锚定 + T-1 落地涨跌，**港股 QDII 可较精准算溢价**；**美股 QDII 当日尚可，次日无法精准**。

## 三、港股 QDII：溢价可较精准测算

**代表基金**：中概互联（164906）、恒生 LOF（501300）等

### 核心优势：交易时间同步

港股与 A 股交易时间高度重合（白天同步开盘、16:00 同步收盘）。当日指数涨跌收盘即落地，模糊空间小。

### 精准计算四步

1. 锚定 **T-2 日官方真实净值** N(T-2)（固定不变）
2. 代入 **T-1 日港股已收盘** 的确定指数涨跌幅 R(T-1)，算出 **T 日基准预估净值**
3. 用 **T 日场内收盘价** P，算出 **当日真实溢价率**
4. T 日港股收盘涨跌 R(T) 落地后，可 **较精准预判次日净值与溢价**

### 小结

| 项目 | 港股 QDII |
|------|-----------|
| 当日溢价 | ✅ 较精准 |
| 次日预估溢价 | ✅ 较精准 |
| LOF Arbiter 标记 | `TRACKING_HK` · 高置信 |

## 四、美股 QDII：只有「预估溢价」，没有精准次日值

**代表基金**：纳指 100 LOF（161130）、标普 500 LOF（161125）、美股科技类 QDII 等

### 核心痛点：完全时差错位

A 股白天交易（9:30–15:00）时，美股处于 **夜间休市**，尚未开盘。

由此产生无法绕过的结构问题：

1. **T 日 A 股收盘时**，美股当日行情完全空白，没有任何落地数据
2. 只能用 **前一晚已收盘的美股涨跌** 计算 T 日溢价（这部分是准的）
3. **次日净值、次日溢价** 只能靠盘前期货、市场情绪预判 → **永远是估算值**

### 关键真相（重点）

| 说法 | 是否正确 |
|------|----------|
| 美股 QDII：**T 日收盘溢价**可较精准（依托前一晚落地行情） | ✅ |
| 美股 QDII：**次日溢价**可精准 | ❌ 100% 预估 |
| APP 显示的「美股基金次日溢价」 | 基于期货动态预判，今晚美股大幅波动后 **直接作废** |

LOF Arbiter 对美股 QDII 标记为 `TRACKING_US`：**仅展示当日可信溢价，不将次日预估作为交易依据。**

## 五、万能计算模板（直接套用）

适配港股、美股跨境 QDII；填空即可。

### 5.1 基础数据

| 符号 | 含义 |
|------|------|
| T | 当日 A 股交易日 |
| N(T-2) | 基金 T-2 日官方净值（唯一精准锚点） |
| R(T-1) | T-1 日海外指数涨跌幅（已收盘真实数据） |
| R(T) | T 日指数涨跌（港股 = 收盘实值；美股盘中 = 不可用/预估） |
| P | 当日场内收盘价（或现价） |

### 5.2 固定公式

```
① T日基准预估净值 = N(T-2) × (1 + R(T-1) / 100)

② 当日溢价率     = (P - 基准预估净值) / 基准预估净值 × 100%

③ T+1预估净值    = 基准预估净值 × (1 + R(T) / 100)

④ 次日预估溢价率 = (P - T+1预估净值) / T+1预估净值 × 100%
```

**LOF Arbiter 扩展**（配置跟踪标的时）：

```
T日基准预估净值 = N(T-2) × (1 + R(T-1)/100 × 仓位) × (1 + 汇率(T-1)/100)
```

## 六、实操案例：164906 中概互联（2026-06-03）

| 输入 | 数值 |
|------|------|
| N(T-2) | 0.9799 |
| R(T-1)，6/2 指数 | +3.86% |
| R(T)，6/3 指数 | -2.5% |
| 场内收盘价 P | 0.988 |

**计算过程：**

```
T日基准预估净值 = 0.9799 × 1.0386 ≈ 1.0177

当日溢价率     = (0.988 - 1.0177) / 1.0177 ≈ -2.92%  （折价）

T+1预估净值    = 1.0177 × 0.975 ≈ 0.9923

次日预估溢价   = (0.988 - 0.9923) / 0.9923 ≈ -0.43%  （折价）
```

> 汇率、仓位、跟踪误差会导致与实盘略有偏差。

**若是纳指类美股 QDII**：当日溢价可较精准；**次日所有数据均为预判，仅作参考，不能作为交易依据。**

## 七、投资避坑准则

### 7.1 港股 QDII

溢价数据 **较为可信**；出现机会可重点把握。

### 7.2 美股 QDII

- **只看当日收盘溢价**（可信）
- **彻底忽略「次日预估溢价」**，不根据预判数据交易
- 盘中实时溢价剧烈波动，**大概率是虚假扰动**（场内情绪 vs 滞后净值）

### 7.3 混合基金（同时含美股 + 港股）

例如「海外科技」类，实际溢价计算 **更为复杂**，较难算准。建议 **T-1 日溢价已经较大** 时再操作，不宜追盘中瞬时数字。

### 7.4 懒人折中

若嫌公式麻烦，可 **直接用当日估算溢价** 决策——毕竟谁也不知道明天场内价格怎么走。但 **绝不要用 APP 的美股「次日预估溢价」下单**。

### 7.5 与 APP「实时溢价」的关系

很多人盲目跟风软件自带「实时溢价」追高买入，结果：

- 收盘溢价一夜抹平，甚至 **大幅折价**
- 隔天直接吃闷亏

根因：**把美股 QDII 的预估值当成精准值**，或 **在 T 日场内价已反映错误预期时追高**。

## 八、一句话总结

| 类型 | 数据闭环 | 当日溢价 | 次日溢价 |
|------|----------|----------|----------|
| **港股 QDII** | 涨跌落地、时差小 | 可较精准计算 | 可较精准预判 |
| **美股 QDII** | 时差阻断 | 可较精准（靠前一晚行情） | 全靠猜，**无精准预判** |

弄懂这套逻辑，就能系统性避开 **溢价套利陷阱** 与 **追高被套** 的大坑。
